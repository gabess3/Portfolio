# Web Design Portfolio

## Description

This web application is a profile for Gabriel Ess as a web developer to display various projects as a showcase for various skills acquired and implemented, such as advanced CSS stylings with flexbox, media queries, psuedo classes, and variables. As more projects are developed, they will be added to this webpage and the webpage will continue to be updated as more skills are acquired.
